# lambada 2.2.0

## Now using the `humane` module for file sizes and relative dates

`humanSize` and the `justincampbell/timeago`-backed `timeAgo` are replaced by [`humane`](https://github.com/woodie/humane), a small formatter module extracted from this exact logic so `lambada` and `scandalous` (via [`humane-ruby`](https://github.com/woodie/humane-ruby)) share one implementation instead of two drifting copies. `humanSize`'s output is unchanged. `timeAgo`'s wording drops the "about" prefix on the hour bucket (`15 hours ago`, not `about 15 hours ago`), matching Finder and the zouk app — the one place `timeago`'s wording diverged from them.

No other user-facing changes.
